# TwoChannelAdapter

The tango-CS server for the "2 channel adapter" PNPI custom device.

Commands:
-----------------------------------
StopMove()      -- stop motion and turn off power <br>
Calibrate()     -- find zero position <br>

Attributes:
-----------------------------------
Position - read the value from encoder and write - set position 

Install:
-----------------------------------
1. mkdir build <br>
2. cd build <br>
3. cmake .. <br>
4. make <br>
